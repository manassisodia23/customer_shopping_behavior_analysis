# 🛍️ Customer Shopping Behavior Analysis

## 📌 Overview
This project is an end-to-end **Data Analytics** solution that analyzes customer shopping behavior using a retail shopping dataset. The workflow covers the complete analytics lifecycle—from data loading and cleaning to exploratory data analysis (EDA), SQL-based business queries, interactive Power BI dashboards, and automated report generation. The objective is to transform raw customer shopping data into meaningful business insights that support data-driven decision-making.

---

## 📂 Dataset
The dataset contains customer shopping transaction records, including information such as:
- Customer Details
- Product Categories
- Purchase Amount
- Quantity
- Payment Method
- Shopping Date
- Location
- Customer Demographics

> **Source:** CSV Dataset

## 📈 Project Workflow

### 1. Data Loading
- Imported the CSV dataset into Python using Pandas.
- Performed initial inspection of the data structure and data types.

### 2. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data formats.
- Standardized column names and data types.

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer demographics.
- Identified purchasing trends.
- Examined sales distribution across product categories.
- Generated visualizations to discover patterns and insights.

### 4. SQL Analysis
- Imported the cleaned dataset into MySQL/SQL Server.
- Executed SQL queries to answer business questions.
- Performed aggregations, filtering, joins, and KPI calculations.

### 5. Power BI Dashboard
Developed an interactive dashboard featuring:
- Sales Overview
- Customer Insights
- Product Category Performance
- Revenue Trends
- Purchase Behavior Analysis
- Interactive Filters and Slicers
- Key Performance Indicators (KPIs)

### 6. Report Generation
- Summarized key business insights.
- Created a professional presentation using **Google Gemini AI**.

## 🚀 How to Run

1. Clone this repository.
   ```bash
   git clone https://github.com/your-username/customer-shopping-behavior-analysis.git
   ```

2. Install the required Python libraries.
   ```bash
   pip install pandas numpy matplotlib seaborn mysql-connector-python
   ```

3. Open the Jupyter Notebook or Python script.

4. Load the CSV dataset.

5. Run the data cleaning and EDA scripts.

6. Import the cleaned data into MySQL or SQL Server.

7. Execute the SQL queries included in the project.

8. Open the Power BI (`.pbix`) file to explore the interactive dashboard.

9. Review the generated report and presentation.
